INSURELINK

A Unified Insurance Operations Platform — Digitizing the Health Insurance Claim Lifecycle by Unifying Insurers, Hospitals, Corporates, Patients, and Admins on One Real-Time Platform

Problem Statement
  • Insurance claim operations in Pakistan remain heavily manual, with hospitals relying on phone calls and emails to coordinate with insurers for approvals.
  • Patients have zero real-time visibility into claim status, approval progress, or remaining coverage balances.
  • Corporate HR departments manage employee insurance data through fragmented spreadsheets with no centralized record-keeping.
  • High turnaround times, frequent communication errors, and complete lack of audit trails plague the current ecosystem.
  • No unified digital platform exists connecting all five stakeholders — insurers, hospitals, corporates, patients, and admins — in one integrated system.

Implementation Overview
Frontend (38+ Pages)
  • 5 Portals: Admin (12 pages), Insurer (8), Hospital (7), Corporate (5), Patient (6)
  • Shared Components: Layouts, sidebars, notification panels, claim tables, messaging UI
  • Real-Time Hooks: useClaimSocket, useNotifications, useGeolocation, useAuth
Backend (50+ REST Endpoints)
  • 12 NestJS Modules with controllers, services, repositories, DTOs
  • Socket.IO Gateway for claim-scoped chat rooms
  • Event-Driven Notifications via EventEmitter2
Security & Compliance
  • JWT Auth: 15-min access / 7-day refresh tokens
  • RBAC: @RolesGuard with cross-org isolation
  • Rate Limiting: 100/min global, 10/min login
  • Audit Trail: @Auditable field-level diffs
Data & Validation
  • Prisma Schema: 15+ entities with relations
  • Pakistan Validation: CNIC, mobile, date formats
  • Bulk Import: CSV/Excel → validate → commit → fix invalid
Key Features & Innovations

5 Role-Based Portals

Real-Time Claims Sync

Socket.IO live status updates, notifications, and cross-tab sync across all stakeholder portals.

Smart Hospital Finder

Leaflet geolocation map with Haversine sorting, reimbursable filtering, and emergency badges.

Bulk Employee Upload
